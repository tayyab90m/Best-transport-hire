# Best Hire Transport - Equipment Rental & General Contracting

Premium construction equipment rental services in UAE.

## Features

- Machinery catalog with availability check
- Online booking system
- WhatsApp integration
- Review system with admin panel
- Dark/Light mode
- Mobile responsive

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Project Structure

```
bvm/
├── app/           # Next.js pages & API routes
├── components/    # React components
├── data/          # JSON data storage
├── lib/           # Utilities & constants
├── public/        # Static assets
│   ├── logo/      # Logo files
│   └── details/   # Machinery images
└── docs/          # Documentation
```

## Tech Stack

- Next.js 16
- Tailwind CSS v4
- JSON file storage (no database)

npm run dev	Start development server (with hot reload)
npm run build	Build for production
npm run start	Start production server (after build)
